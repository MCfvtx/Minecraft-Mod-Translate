# Minecraft模组汉化
本项目灵感来自943大佬的汉化Mod，然而汉化Mod目前只在汉化1.12.2版本的Mod，新版本并不在参与，所以本项目致力于1.15.2及以上版本Mod的汉化，因技术力有限，并没有开发一个自动更新的Mod，所以需要您的自行解包添加，放心，这并不难，感谢您对这个项目的支持
